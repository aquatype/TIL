# Python to JSON Encoding Table

| Python            | JSON       |
| :---------------- | ---------- |
| dict {}           | object     |
| list [], tuple () | array      |
| str, unicode      | string     |
| int, long, float  | number     |
| True, False       | true, false |
| None              | null       |
